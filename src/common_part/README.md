# 通用内容

## 核心点和通用点

不同可执行文件格式的内部，最核心的是：

* 编译和链接的过程
  * 涉及到编译器和链接器
    * 如何生成代码？
    * 把生成的代码和数据，放到什么地方？
    * 运行时如何加载这些代码和数据？
      * 才能确保程序能正常执行

## 不用文件格式的通用内容

* 各种的
  * Segment段
    * 代码段
      * `.text`
    * 数据段
      * `.data`
    * 等等
  * section区
    * `.bss`
    * `.data`
    * `.rodata`
    * 等等
